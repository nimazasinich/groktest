# راهنمای نصب و استفاده از برنامه تشخیص گفتار فارسی

## چگونگی راه‌اندازی پروژه

با این مراحل می‌توانید به راحتی پروژه را راه‌اندازی کنید:

### مرحله 1: ایجاد پوشه‌ها

1. یک پوشه اصلی به نام `speech-to-text-app` ایجاد کنید
2. در داخل آن این پوشه‌ها را ایجاد کنید:
   - `src`
   - `src/renderer`
   - `assets`

### مرحله 2: ذخیره فایل‌ها

هر فایل را که دانلود کرده‌اید در مسیر مربوطه ذخیره کنید:

- `package.json` → در پوشه اصلی
- `webpack.config.js` → در پوشه اصلی
- `postcss.config.js` → در پوشه اصلی
- `tailwind.config.js` → در پوشه اصلی
- `main.js` → در پوشه `src`
- `preload.js` → در پوشه `src`
- `index.html` → در پوشه `src/renderer`
- `index.js` → در پوشه `src/renderer`
- `index.css` → در پوشه `src/renderer`
- `SpeechToTextUIWithTranslation.jsx` → در پوشه `src/renderer`

### مرحله 3: دانلود آیکون

1. یک آیکون میکروفون دانلود کنید (می‌توانید از [flaticon.com](https://www.flaticon.com/free-icon/microphone_25682) استفاده کنید)
2. آن را با نام `microphone.png` در پوشه `assets` ذخیره کنید

### مرحله 4: نصب وابستگی‌ها

1. خط فرمان (Command Prompt یا PowerShell) را باز کنید
2. به پوشه پروژه بروید
   ```
   cd مسیر\به\پوشه\speech-to-text-app
   ```
3. این دستور را اجرا کنید:
   ```
   npm install
   ```

### مرحله 5: اجرای برنامه در حالت توسعه

برای آزمایش برنامه قبل از ساخت نسخه نهایی:
```
npm start
```

### مرحله 6: ساخت نسخه قابل حمل (EXE)

برای ساخت فایل اجرایی واحد:
```
npm run build
npm run make:portable
```

فایل اجرایی در پوشه `dist` با نام `SpeechToTextApp.exe` ایجاد می‌شود.

## نکات مهم

1. **مشکل نصب robotjs**: 
   - اگر در مرحله 4 با خطای نصب روبرو شدید، نگران نباشید
   - فایل‌های این پروژه به گونه‌ای تغییر یافته‌اند که به robotjs نیازی ندارند
   - به جای شبیه‌سازی کلیدها، متن در کلیپ‌بورد کپی می‌شود و می‌توانید با Ctrl+V آن را درج کنید

2. **تشخیص گفتار فارسی**:
   - این برنامه از Web Speech API استفاده می‌کند
   - به صورت پیش‌فرض زبان فارسی را پشتیبانی می‌کند
   - در صورتی که Windows شما زبان فارسی را پشتیبانی نمی‌کند، باید زبان فارسی را به آن اضافه کنید

3. **نحوه استفاده**:
   - با کلیک روی دکمه میکروفون یا فشردن Alt+Space، تشخیص گفتار را شروع کنید
   - برای مشاهده ترجمه انگلیسی، روی فلش کوچک در زیر متن کلیک کنید
   - برای کپی متن، روی دکمه کپی کلیک کنید
   - برای درج متن، روی دکمه درج کلیک کنید و سپس Ctrl+V را در محل مورد نظر فشار دهید
   - متن‌های مفید را با دکمه ذخیره، ذخیره کنید

4. **حالت مینیمایز**:
   - با کلیک روی دکمه مینیمایز در بالای صفحه، برنامه به حالت کوچک در می‌آید
   - برای بازگشت به حالت عادی، روی دکمه ماکسیمایز کلیک کنید

## نیازمندی‌های سیستم

- Windows 10 یا بالاتر
- 4GB RAM یا بیشتر
- میکروفون
- یک مرورگر مدرن (برای پشتیبانی از Web Speech API)

## رفع مشکلات متداول

- **برنامه اجرا نمی‌شود**: مطمئن شوید Node.js نسخه 14 یا بالاتر نصب شده باشد
- **تشخیص گفتار کار نمی‌کند**: مطمئن شوید میکروفون کامپیوتر درست کار می‌کند و در Windows مجوز دسترسی میکروفون را داده‌اید
- **تشخیص گفتار فارسی ضعیف است**: مطمئن شوید زبان فارسی در Windows شما نصب شده است
- **متن درج نمی‌شود**: پس از کلیک روی دکمه درج، باید خودتان Ctrl+V را فشار دهید
